# foldmold
mold with a fold

Instructions:
1. Install Blender
2. Zip up the foldmold directory
1. Open Edit->Preferences->Add-ons
2. Click on "Install..." and select the zip file
3. Tick the checkbox to enable the addon


Python build:

[For Mac](./install_bpy_mac.md)
